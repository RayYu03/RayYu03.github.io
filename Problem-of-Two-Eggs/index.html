Problem of Two Eggs

Posted on 2017-03-25   |  

赶火车之前看了一眼腾讯2018年暑假实习生的笔试题，其中一道就是Problem of Two Eggs的变形……

还是简单说下自己的思路吧。

题目

一幢 100 层的大楼，给你两个鸡蛋，如果在第 n 层扔下鸡蛋，鸡蛋不碎，那么从第 n-1 层扔鸡蛋，都不碎。这两只鸡蛋一模一样，不碎的话可以扔无数次，且鸡蛋在0层不会碎。设计一种策略能保证可以测出鸡蛋恰好会碎的楼层，且如果该策略是最坏的情况所扔次数最少。

思路

整个题目中最最最最重要的就是最后一句话。

最坏的情况所扔次数最少

两种情况

假设从第 k 层开始往下扔:

  • 没有坏，我们可以继续从 [k + 1, 100] 这个区间内继续扔了。
  • 如果坏了，那么我们就只有一个鸡蛋了，我们需要从 [1, k - 1] 这个区间内继续扔鸡蛋了。这个时候就是最坏的情况，即需要扔 k 次。

鸡蛋没有坏

如果鸡蛋没有坏，我们可以继续收敛结果。

可以分为以下三种情况：

  • 增加 k 层
  • 增加 k + x 层
  • 增加 k - x 层

0 < x < k

第一种情况

如果增加 k 层碎了的话， 则最坏情况为 2k - (k - 1) + 2 = k + 1 > k。此时比第一种情况更坏，以后也会越来越坏。所以这个策略舍弃。

第二种情况

如果增加 k + x 层碎了的话，和第一种情况一样，以后也会越来越坏。所以这个策略舍弃。

第三种情况

如果增加 k - x 层碎了的话， 则最坏情况为 (2k - x) - (k - 1) + 2 = k - x < k。 随着 k 的不断减小，最坏情况始终为 k。其他情况均比 k 小。

即满足了 最坏的情况所扔次数最少。

此时我们需要如何使扔的次数最少，若想覆盖到所有情况，则需要设置 x = 1.

即

如果第 k 层没有碎，则增加 k - 1 层；
如果第 k + k - 1 层没有碎，则增加 k - 2 层；

….

以此类推。

得出数学公式：

k + (k - 1) + (k - 2) + … + 2 + 1 = k(k + 1) / 2

如果扔100次， 则：

k(k + 1) / 2 = 100 => x ≈ 14
