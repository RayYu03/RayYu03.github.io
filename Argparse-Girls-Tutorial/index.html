Argparse Girls Tutorial

Argparse 版本号： 3.2.
Python 版本号： 3.6.0

本教程旨在简单的介绍一下argparse的用法，这是Python标准库中推荐的命令行解析模块。

注意：有两个完成相同任务的其他模块，即getopt（C语言的getopt()的等价物）和已弃用的optparse。还要注意，argparse基于optparse，因此两者在使用方面非常相似。

概念

首先使用 ls 命令来展示我们将在本入门教程中探索的功能类型：

我们可以从四个命令学习一些概念：

  • ls命令在没有任何选项的情况下很有用。它默认显示当前目录的内容。
  • 如果我们想要比它默认提供的功能。比如，我们希望它显示一个不同的目录 pypy。我们所做的就是指定位置参数。仅仅基于它在命令行上出现的位置，就知道如何处理值。这个概念与cp命令更相似，cp的最基本用法是cp SRC DEST。第一个位置是您要复制的位置，第二个位置是您希望复制到的位置。
  • 如果我们想改变程序的行为。比如，我们为每个文件显示更多信息，而不仅仅显示文件名。在这种情况下的-l被称为可选参数。
  • 这是帮助文本的一段。如果你遇到一个你从来没有使用过的程序，可以通过阅读它的帮助文本，弄清楚它的工作原理。
